pagetox-ruby
============

ruby api for html to image and pdf conversion

###Installation###
```
bundle install
```
###2nd Dependencies###
copy binaries from lib folder to /usr/local/bin/
```
cp ./lib/* /usr/local/bin/
```

###Run on Thin###
```
ruby pagetox.rb
```

###Run with Puma ###
just type 
```
puma
```
